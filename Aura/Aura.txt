1. 项目概述 (Project Overview)
1.1. 产品愿景 (Vision)
打造一款名为"Aura小记"的应用，其设计美学与用户体验对标苹果官网水准。它通过四个核心功能模块（记事、待办、AI对话、番茄钟）帮助用户高效管理任务与时间，并通过高度可定化的视觉主题（如极简苹果、多巴胺、糖果渐变）为用户提供愉悦和个性化的使用体验。

1.2. 目标用户 (Target Audience)

学生、自由职业者、程序员、设计师等需要管理任务和时间的专业人士。
注重软件界面美观和用户体验的群体。
喜欢通过个性化设置彰显个性的用户。
1.3. 核心价值主张 (Core Value Proposition)

极致体验： 流畅、直观、视觉上令人愉悦的单页面应用。
功能聚焦： 四大核心功能，解决常见生产力痛点，避免功能臃肿。
个性表达： 独特的"主题调色盘"功能，让应用不仅是应用，更是个人风格的延伸。
2. 用户角色 (User Personas)
学生-小陈： 需要记录课堂笔记、规划作业截止日期，并使用番茄钟法保持专注。喜欢清新明亮的界面。
设计师-小林： 对UI美学有极高要求，会用待办事项管理项目进度，并需要与AI对话获取灵感。偏爱极简和高级的设计。
开发工程师-小王： 使用番茄钟进行深度编程，需要记事本记录临时想法和代码片段。欣赏高效、可定制的应用。
3. 功能需求详述 (Detailed Functional Requirements)
3.1. 全局特性 (Global Features)

G1: SPA (单页面应用) 导航
描述：应用应实现无页面刷新的流畅导航，通过顶部导航栏在四个模块间切换。
验收标准：点击导航栏链接，内容区域平滑切换，URL同步更新，浏览器前进/后退功能正常工作。
G2: 统一的设计语言 (Apple-Inspired)
描述：整个应用遵循苹果设计哲学：充足留白、简约布局、圆角设计、细腻字体、克制动效。
验收标准：界面视觉评估与苹果官网风格一致，用户体验感到“高级”和“舒适”。
G3: 响应式设计
描述：界面应良好适配从手机到桌面电脑的不同屏幕尺寸。
验收标准：在主流移动设备和桌面浏览器上，布局正常，功能可用。
3.2. 模块 1: 记事 (Notes)

N1: 笔记列表视图
描述：以卡片列表形式展示所有已创建的笔记，显示标题和部分预览内容。
N2: 创建与编辑笔记
描述：提供富文本编辑器（至少支持纯文本），允许用户创建新笔记和编辑现有笔记。
N3: 持久化存储
描述：所有笔记内容应自动保存到浏览器的本地存储（localStorage）或后续的数据库中。
N4: 删除笔记
描述：提供删除笔记的功能，操作前应有确认提示。
3.3. 模块 2: 待办事项 (Todo)

T1: 任务列表视图
描述：展示所有待办任务，清晰区分“未完成”和“已完成”状态。
T2: 添加新任务
描述：通过输入框快速添加新的待办任务。
T3: 标记完成/未完成
描述：通过复选框轻松标记任务的完成状态。完成的任务应有视觉变化（如删除线）。
T4: 删除任务
描述：提供删除单个任务的功能。
3.4. 模块 3: AI对话 (Chat)

C1: 对话界面
描述：呈现一个类聊天应用的界面，包含消息历史区域和底部的输入框。
C2: 发送消息
描述：用户输入文本并发送后，消息应显示在聊天区域。
C3: 模拟AI回复 (V1阶段)
描述：V1版本可模拟AI回复，例如返回一条固定的欢迎信息或回显用户的消息。为后续集成真实AI API（如OpenAI）预留接口。
C4: 清空对话
描述：提供“新对话”按钮，用于清空当前对话历史。
3.5. 模块 4: 番茄钟 (Pomodoro) - 核心特色模块

P1: 计时器功能
描述：实现标准的番茄工作法计时器（例如：25分钟工作，5分钟休息），具备开始、暂停、重置等控制按钮。
验收标准：计时准确，计时结束时应有明显的视觉或声音提示。
P2: 主题调色盘 (核心功能)
描述：提供主题切换控件，允许用户在三种预设视觉主题间动态切换：
P2.1 极简-苹果风格: 默认主题。中性色（白、灰）、大量留白、细腻字体。
P2.2 多巴胺风格: 高饱和度、高对比度的鲜艳色彩，充满活力。
P2.3 糖果渐变风格: 使用明亮、甜美的颜色（粉、蓝、绿）构建的柔和渐变背景。
验收标准：切换主题时，界面颜色、背景等样式立即无刷新更新。
P3: 状态持久化
描述：用户选择的主题偏好应自动保存到localStorage，下次访问时自动应用。
4. 非功能性需求 (Non-Functional Requirements)
性能： 应用加载速度快，界面交互流畅，无卡顿。
可用性： 界面直观，新用户无需教程即可上手。导航清晰，功能易于发现。
兼容性： 支持最新版本的Chrome, Firefox, Safari, Edge浏览器。
可维护性： 代码结构清晰，注释良好，组件化程度高，便于后续功能扩展。
5. 技术栈 (Tech Stack)
前端框架: Vue 3 (Composition API + <script setup>)
构建: Vite
路由: Vue Router
样式/UI:
Tailwind CSS
Shadcn/UI for Vue (组件库)
Lucide (图标库)
状态管理 (模块内): Vue Reactive API (ref, reactive)
数据持久化 (前端): localStorage
语言: TypeScript (推荐) / JavaScript
6. 后续版本规划 (Future Considerations)
用户系统: 实现注册/登录，支持数据云同步。
真实AI集成: 接入大型语言模型（如GPT-4）API，提供智能对话和笔记辅助功能。
高级番茄钟: 可自定义工作时长/休息时长，统计历史专注数据。
更多主题: 允许用户自定义颜色或添加更多预设主题。
PWA支持: 将应用包装为渐进式Web应用，可安装到桌面。
